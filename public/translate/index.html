How to contribute

1. Sign Up for Crowdin

Create a free Crowdin account. It only takes a minute and gives you access to translate Morphe and other open source projects.

2. Choose Your Language

Select the language you want to translate into. You can contribute to new translations or improve existing ones.

3. Start Translating

Use Crowdin's intuitive interface to translate strings. Every contribution helps make Morphe accessible to more users.

4. See Your Impact

Your translations will be reviewed and included in future Morphe releases, helping users around the world.
